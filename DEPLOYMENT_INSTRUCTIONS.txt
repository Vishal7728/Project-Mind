STEP 3: Build APK
─────────────────
The deploy_apk.py script will:
✓ Verify Android SDK installation

✓ Create release APK (5-15 minutes)

STEP 4: Connect Android Device
───────────────────────────────
Required setup:
1. Connect Android device via USB
2. Enable Developer Mode (Settings > About > Build Number x7)
3. Enable USB Debugging (Settings > Developer Options > USB Debugging)
4. Allow USB debugging from this PC (device prompt)

STEP 5: Automatic Installation
────────────────────────────────
The deploy_apk.py script will:
✓ Detect connected device
✓ Install APK automatically
✓ Launch the application
✓ Confirm successful installation

TROUBLESHOOTING
═══════════════

If Android SDK not found:
  → Download Android Studio from developer.android.com
  → Or install Android SDK command-line tools
  → Then set ANDROID_HOME environment variable

If Java not found:
  → Download Java JDK 17+ from adoptopenjdk.net
  → Add to PATH environment variable
  → Restart PowerShell

If device not detected:
  → Check USB cable connection
  → Enable USB debugging on device
  → Run: adb devices
  → Should show device serial number

If APK installation fails:
  → Check device has 50MB+ free storage
  → Uninstall previous version: adb uninstall org.projectmind.projectmind
  → Retry installation
