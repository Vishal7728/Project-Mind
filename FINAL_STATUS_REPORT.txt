TROUBLESHOOTING GUIDE
═══════════════════════════════════════════════════════════════════════════════

"Buildozer not found on Windows"
─────────────────────────────────
Solution: Windows doesn't support native Android builds
Use: Docker, WSL2, or GitHub Actions instead

"adb not found"
─────────────────────────────────
Solution 1: Add to PATH:
  C:\Users\Vishal Kumar\AppData\Local\Android\Sdk\platform-tools

Solution 2: Use full path:
  C:\Users\Vishal Kumar\AppData\Local\Android\Sdk\platform-tools\adb.exe

"App crashes on device"
─────────────────────────────────
Check logs: adb logcat | grep python
Ensure: Android 13+ installed on device
Verify: All permissions granted to app

"Build takes >30 minutes"
─────────────────────────────────
Normal: First build is slower (downloads SDK, dependencies)
Subsequent builds are faster (cached dependencies)
Ensure: 10-15GB free disk space available

"Permission denied errors"
─────────────────────────────────
WSL2: Run with elevated privileges
Docker: Ensure Docker is running
Linux: Check file permissions
